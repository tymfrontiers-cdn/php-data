# PHP Data manager

Manage data, encryption and more.

## Requirement

- PHP 5.5 and highier

## Installation

Using composer:

`$ composer require tymfrontiers-cdn/php-data`

## Usage

```
use TymFrontiers;

```
Methods are self-explanatory, kindly check [Class file](src/Data.php).
